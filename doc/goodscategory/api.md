
# goods_category 商品规格分类模块 API
## 1.1 查询商品规格分类详情

> **描述**：根据主键ID查询商品规格分类详情。

> **url**：/admin/goods_category_rest/get/{id}
>
> **method**：GET

> **输入**

| 序号 | 字段名称 | 字段类型 | 字段描述     |
| ---- | -------- | -------- | ------------ |
| 1    | id       | Long     | 主键ID，必填 |

> **输出**

| 序号     | 字段名称      | 字段类型 | 字段描述                |
| -------- | ------------- | -------- | ----------------------- |
| 1        | flag          | boolean  | 接口调用，成功/失败标识 |
| 2        | code          | int      | 错误码，成功时=0        |
| 3        | desc          | string   | 结果描述                |
| 4        | data          | object   | 商品规格分类详情对象            |
|          |               |          |                         |
| 字段解释 |               |          |                         |
| 1 | id              | Long          | 主键 |
| 2 | name              | String          | 分类名称 |
| 3 | keyword              | String          | 分类另一个关键词 |

## 1.6 分页查询商品规格分类

> **描述**：分页查询商品规格分类。

> **url**：/admin/goods_category_rest/page
>
> **method**：GET

> **输入**

| ---- | -------- | -------- | ------------ |
| 1    |page      |int       |第几页 [1,n)   |

| 3    |safeOrderBy      |int       | 排序 例如 数据库字段名称 desc或asc   |

| 2        | code          | int      | 错误码，成功时=0        |
| 3        | desc          | string   | 结果描述                |
| 4        | data          | array   | 商品规格分类详情数组对象            |
|          |               |          |                         |
| 字段解释 |               |          |                         |
